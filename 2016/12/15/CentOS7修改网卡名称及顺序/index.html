CentOS7修改网卡名称及顺序

发表于 2016-12-15   |   分类于 LinuxSystem   |  

CentOS7下，网卡名称有时是一些任意的类似eno166666或p3p1等样式的名字。为了方便记忆与使用，有时需要将网卡名称修改为传统的ethX格式，以下内容为修改网卡名称及网卡顺序的方法。

在CentOS7系统下，修改网卡名称主要由以下几步：

  1. 修改内核启动参数
  2. 修改udev rules脚本
  3. 重命名并修改/etc/sysconfig/network-scripts/下的接口配置脚本
  4. 重启设备，确认是否修改成功

修改内核启动参数

为了使网卡名称恢复到旧的网卡名称（ethX），需要修改/etc/default/grub文件的GRUB_CMDLINE_LINUX内容，增加net.ifnames=0，修改后内容如下：

[root@localhost ~]# cat /etc/default/grub 
GRUB_TIMEOUT=5
GRUB_DISTRIBUTOR="$(sed 's, release .*$,,g' /etc/system-release)"
GRUB_DEFAULT=saved
GRUB_DISABLE_SUBMENU=true
GRUB_TERMINAL_OUTPUT="console"
GRUB_CMDLINE_LINUX="crashkernel=auto rd.lvm.lv=centos00/root rd.lvm.lv=centos00/swap \
            rhgb net.ifnames=0 quiet"
GRUB_DISABLE_RECOVERY="true"

更新内核启动参数：

  1. 如果系统是以BIOS方式启动:

    grub2-mkconfig -o /boot/grub2/grub.cfg

  2. 如果系统是以UEFI方式启动：

    grub2-mkconfig -o /boot/efi/EFI/centos/grub.cfg

注意：在一些网络文章里面给出的内核参数为net.ifnames=0 biosdevname=0，经验证发现加入biosdevname=0参数后网卡名称确实会被修改为ethX，但是无法修改MAC地址对应的网卡名称，即无法调整网卡顺序。

修改udev rules文件

在/usr/lib/udev/rules.d/60-net.rules文件中将原有内容删除，然后将每块网卡对应的MAC地址及修改后的名称依次写入/usr/lib/udev/rules.d/60-net.rules文件，例如：将MAC地址为a4:ba:db:21:55:25的网卡名称修改为eth100，只需要在上述文件中增加一行：ACTION=="add", SUBSYSTEM=="net", DRIVERS=="?*", ATTR{type}=="1", ATTR{address}=="a4:ba:db:21:55:25", NAME="eth100"即可。以下为一个修改后的脚本样例：

ACTION=="add", SUBSYSTEM=="net", DRIVERS=="?*", ATTR{type}=="1", \
       ATTR{address}=="a4:ba:db:21:55:25", NAME="eth0"
ACTION=="add", SUBSYSTEM=="net", DRIVERS=="?*", ATTR{type}=="1", \
       ATTR{address}=="2c:53:4a:01:62:88", NAME="eth1"
ACTION=="add", SUBSYSTEM=="net", DRIVERS=="?*", ATTR{type}=="1", \
       ATTR{address}=="2c:53:4a:01:62:89", NAME="eth2"
ACTION=="add", SUBSYSTEM=="net", DRIVERS=="?*", ATTR{type}=="1", \
       ATTR{address}=="a4:ba:db:21:55:2b", NAME="eth3"
ACTION=="add", SUBSYSTEM=="net", DRIVERS=="?*", ATTR{type}=="1", \
       ATTR{address}=="a4:ba:db:21:55:27", NAME="eth4"
ACTION=="add", SUBSYSTEM=="net", DRIVERS=="?*", ATTR{type}=="1", \
       ATTR{address}=="a4:ba:db:21:55:29", NAME="eth5"

修改网卡配置文件

修改/etc/sysconfig/network-scripts/目录下的各个网卡的配置。
将相对应的网卡的名称如：ifcfg-em1修改为ifcfg-eth0，并修改配置脚本里面的NAME和DEVICE字段为eth0。

重启设备

设备重启后，使用ifconfig命令查看对应的MAC地址与接口名称，确认是否修改成功。
